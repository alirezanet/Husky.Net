## Installation

```shell
dotnet tool install --global Husky
```

## Setup husky for your project

```shell
cd <Your project root directory> # <-- important
husky install
```

## Add your first hook

```shell
husky add .husky/pre-commit "echo 'Husky is awesome!'"
```
